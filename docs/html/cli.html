Cylleneus¶

Corpus management¶

cylleneus¶

Indexing commands for Cylleneus.

cylleneus [OPTIONS] COMMAND [ARGS]...

add¶

Index a specific file.

cylleneus add [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-p, --path <path>¶

[required]

-a, --author <author>¶
-t, --title <title>¶

clear¶

Clear the indexes of a corpus without deleting them.

cylleneus clear [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

create¶

Create all corpus indexes from source files.

cylleneus create [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-d, --destructive, -D, --not-destructive¶
-o, --optimize¶

delete¶

Delete a document in a corpus by index number.

cylleneus delete [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-d, --docix <docix>¶

[required]

delete-by¶

Delete documents in a corpus by author name and/or title.

cylleneus delete-by [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-a, --author <author>¶
-t, --title <title>¶

destroy¶

Destroy the indexes and manifest of a corpus.

cylleneus destroy [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

download¶

Download a remote corpus repository.

cylleneus download [OPTIONS]

Options

-c, --corpus <corpus>¶
-b, --branch <branch>¶

download-by¶

Download documents by author and title from a remote corpus.

cylleneus download-by [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-a, --author <author>¶
-t, --title <title>¶

download-by-docix¶

Download a document by number from a remote corpus.

cylleneus download-by-docix [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-i, --docix <docix>¶

index¶

List indexed works in a corpus.

cylleneus index [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

lexicon¶

List the contents of an index by field name.

cylleneus lexicon [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-f, --fieldname <fieldname>¶

[required]

optimize¶

Optimize the indexes of a corpus.

cylleneus optimize [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

update¶

Reindex a document by index number.

cylleneus update [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-d, --docix <docix>¶

[required]

-p, --path <path>¶

[required]

update-by¶

Reindex a document by author name and/or title.

cylleneus update-by [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-a, --author <author>¶
-t, --title <title>¶
-p, --path <path>¶

[required]

verify¶

Verify the integrity of corpus indexes and manifest.

cylleneus verify [OPTIONS]

Options

-c, --corpus <corpus>¶

[required]

-v, --verbose¶
-d, --dry-run¶

version¶

Display the current engine version number.

cylleneus version [OPTIONS]

where¶

Display location of corpus directory.

cylleneus where [OPTIONS]
